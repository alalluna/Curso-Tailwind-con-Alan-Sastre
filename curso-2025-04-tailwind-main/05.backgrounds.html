Backgrounds en Tailwind CSS

1. Background attachment

Controla cómo se comporta una imagen de fondo al hacer scroll.

bg-fixed

El fondo es fijo con respecto al viewport.

Scrollable content...

bg-local

El fondo se desplaza con el contenido del elemento.

Scrollable content...

bg-scroll (default)

El fondo es fijo con respecto al elemento, no se desplaza con el contenido.

Scrollable content...

2. Background Clip

Controla hasta dónde se extiende el fondo dentro de un elemento.

bg-clip-border (default) El fondo se extiende hasta el borde exterior.

bg-clip-padding El fondo se extiende hasta el borde interior del elemento.

bg-clip-content El fondo se recorta (clip) para ajustarse al área de contenido.

bg-clip-text
