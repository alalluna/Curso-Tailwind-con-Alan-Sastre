Guía de Dark Mode con Tailwind CSS 4.1

Selector de Tema

¿Cómo usar Dark Mode en Tailwind CSS 4.1?

Tailwind CSS 4.1 facilita la implementación del modo oscuro con el prefijo dark: para cualquier utilidad.

<div class="bg-white dark:bg-slate-800 text-black dark:text-white">
  Este div cambiará de estilo en modo oscuro
</div>

Implementación del Selector de Tema

Para implementar un selector de tema como el de arriba, necesitas:

  1. Agregar/quitar la clase dark al elemento HTML
  2. Almacenar la preferencia en localStorage
  3. Detectar la preferencia del sistema cuando se selecciona "Tema Sistema"
// Verificar preferencia guardada o usar preferencia del SO
if (localStorage.theme === 'dark' || 
   (!('theme' in localStorage) && 
    window.matchMedia('(prefers-color-scheme: dark)').matches)) {
  document.documentElement.classList.add('dark')
} else {
  document.documentElement.classList.remove('dark')
}

// Función para cambiar el tema
function setTheme(theme) {
  if (theme === 'dark') {
    document.documentElement.classList.add('dark')
    localStorage.theme = 'dark'
  } else if (theme === 'light') {
    document.documentElement.classList.remove('dark')
    localStorage.theme = 'light'
  } else {
    localStorage.removeItem('theme')
    // Aplicar tema según SO
  }
}
