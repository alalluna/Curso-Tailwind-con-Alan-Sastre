¿Qué es Flexbox?

Flexbox es un modelo de layout unidimensional: distribuye espacio a lo largo de un eje principal y uno cruzado. Con display: flex (o inline-flex) conviertes un contenedor en un flex container; sus hijos pasan a ser flex items. Tailwind CSS expone cada ajuste como una clase utilitaria — sin escribir CSS manual.

Esta guía recorre todas las utilidades de Flexbox en Tailwind 4.1, con explicaciones y ejemplos listos para copiar y pegar.

1. Activar Flexbox — flex / inline-flex

Estas dos clases cambian la propiedad display:

  • flex → display: flex;
  • inline-flex → display: inline-flex;

Ejemplo con flexbox:

2. Dirección del eje principal — flex-direction

Controla si los ítems se apilan en fila o columna (y su orden):

Filas

  • flex-row (default)
  • flex-row-reverse

Columnas

  • flex-col
  • flex-col-reverse

3. Ajuste de línea — flex-wrap

Permite que los ítems pasen a la línea siguiente cuando no caben:

  • flex-wrap
  • flex-wrap-reverse
  • flex-nowrap (default)
<div class="flex flex-wrap gap-2">
  <div class="w-24 h-10 bg-sky-500"></div>
  ...
</div>

4. Tamaño inicial — flex-basis

Define el tamaño base de un ítem antes de que flex-grow/flex-shrink entren en juego. Tailwind provee utilidades basis-{valor} que aceptan tanto unidades fijas (basis-24) como fracciones (basis-1/3), porcentajes o auto.

5. Crecimiento — flex-grow

Permite que un ítem tome el espacio libre disponible. Tailwind utiliza:

  • grow → flex-grow: 1
  • grow-0 → flex-grow: 0 (default)

6. Contracción — flex-shrink

Indica si el ítem puede reducirse cuando el contenedor no tiene espacio suficiente:

  • shrink → flex-shrink: 1 (default)
  • shrink-0 → flex-shrink: 0

12. Shorthand — place-content, place-items, place-self

Las utilidades place-* son atajos para establecer la alineación en ambos ejes a la vez.

  • place-content-center: equivale a justify-content: center; + align-content: center;
  • place-items-end: atajo de justify-items + align-items
  • place-self-start: atajo de justify-self + align-self
